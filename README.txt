# Mini Mercado Sousa

Este projeto é um sistema web funcional para gerenciamento de produtos e compras de um mini mercado. Ele inclui páginas para diferentes categorias de produtos, carrinho de compras, cadastro de clientes e serviços. As funcionalidades interativas garantem uma experiência simples e intuitiva para o usuário.

---

## **Funcionalidades**

### **1. Navegação**
- **Páginas disponíveis**:
  - **Página inicial (`index.html`)**: Apresentação geral do mercado com carrossel de imagens.
  - **Alimentos (`alimentos.html`)**: Lista de produtos alimentícios.
  - **Frutas e Verduras (`frutas-verduras.html`)**: Lista de frutas e vegetais.
  - **Higiene (`higiene.html`)**: Produtos de higiene pessoal e limpeza.
  - **Serviços (`servicos.html`)**: Informações sobre os serviços oferecidos.
  - **Carrinho (`carrinho.html`)**: Controle de itens adicionados ao carrinho.
  - **Cadastro (`cadastro.html`)**: Cadastro e login de clientes.
  - **Agendamento (`agendamento.html`)**: Formulário para agendamento de serviços.

### **2. Carrinho de Compras**
- Adição de produtos ao carrinho diretamente das páginas de produtos.
- Exibição de:
  - Miniaturas das imagens dos produtos.
  - Nome, quantidade e preço subtotal de cada item.
  - Valor total da compra.
- Opções para ajustar a quantidade de itens no carrinho.
- Confirmação de compra com mensagem interativa.

### **3. Cadastro e Agendamento**
- Formulário para cadastro de novos clientes com simulação de envio.
- Simulação de login para usuários já cadastrados.
- Formulário para agendamento de serviços com feedback de confirmação.

### **4. Responsividade**
- Layout adaptado para dispositivos móveis, tablets e desktops, otimizando a experiência do usuário.

### **5. Acessibilidade**
- Design pensado para inclusão e fácil navegação, com contrastes de cores e botões intuitivos.

## **Estrutura do Projeto**

### **Páginas HTML**
- **`index.html`**: Página inicial com carrossel e navegação entre as categorias.
- **`alimentos.html`**: Produtos alimentícios.
- **`frutas-verduras.html`**: Frutas e verduras.
- **`higiene.html`**: Produtos de higiene.
- **`servicos.html`**: Descrição dos serviços.
- **`carrinho.html`**: Itens adicionados e total da compra.
- **`cadastro.html`**: Cadastro de clientes.
- **`agendamento.html`**: Formulário de agendamento de serviços.

### **Estilos**
- **`style.css`**: Folha de estilo principal com ajustes visuais e responsividade.

### **Scripts**
- **`script.js`**: Gerenciamento das interações do carrinho, navegação e formulários.

### **Dados**
- **`produtos.json`**: Base de dados dos produtos exibidos no site.

---

## **Tecnologias Utilizadas**
- **HTML5**: Estrutura das páginas.
- **CSS3**: Design e estilização.
- **JavaScript**: Funcionalidades e interatividade.
- **JSON**: Gerenciamento e exibição de dados.
- **Bootstrap 5**: Componentes responsivos e design moderno.
